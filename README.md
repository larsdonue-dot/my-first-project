## 📧 Как работает форма

```
Пользователь заполняет форму на сайте
        ↓
JavaScript обрабатывает и отправляет данные
        ↓
Netlify Function отправляет в Telegram
        ↓
Netlify Forms сохраняет заявку
        ↓
Email с заявкой приходит на zmitro@mail.ru
```

**Результат:** Вы получаете уведомления тремя способами:
1. 📧 Email (автоматический, через Netlify)
2. 💬 Telegram (через Netlify Function)
3. 📋 Submissions в Netlify Forms

## 🚀 Развёртывание

### Требования
- GitHub аккаунт (бесплатный)
- Netlify аккаунт (бесплатный)
- Telegram аккаунт (для бота)

### Пошагово

1. **Подготовка:**
   ```bash
   git add .
   git commit -m "Initial commit"
   git push origin main
   ```

2. **На Netlify:**
   - Откройте netlify.com
   - Импортируйте GitHub репо
   - Нажмите "Deploy"

3. **Переменные окружения:**
   - Netlify → Site settings → Environment
   - Добавьте `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID`

4. **Email уведомления:**
   - Netlify → Forms → contact → Notifications
   - Добавьте zmitro@mail.ru

**Готово!** 🎉

## ❓ FAQ

**Q: Могу ли я изменить дизайн?**
A: Да! Всё в CSS, легко менять цвета, шрифты, layout.

**Q: Что если я не разработчик?**
A: Достаточно обновить текст, фото и информацию о проектах.

**Q: Сколько стоит Netlify?**
A: Free tier достаточен для этого сайта (100 GB bandwidth, 125K функций в месяц).

**Q: Надо ли платить за Telegram?**
A: Нет, Telegram Bot API бесплатный.

**Q: Как добавить свои проекты?**
A: Отредактируйте раздел "Мои проекты" в index.html.

**Q: Могу ли я отправлять заявки в несколько мест?**
A: Да, добавьте дополнительные интеграции в send-telegram.js.
